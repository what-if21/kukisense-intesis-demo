## Key Memory
The GDEP073E01 requires this exact display sequence:
1. DTM (0x10) - send ALL pixel data first
2. PON (0x04) - power on
3. BTST2 (0x06) - booster soft start
4. DRF (0x12) - refresh
5. Wait BUSY
6. POF (0x02) - power off
