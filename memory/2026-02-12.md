# 2026-02-12 - E-Ink Frame Project Fixes

## GDEP073E01 (Spectra 6) Display Fixes

### PSR (Panel Setting) Values
Correct values for 800x480, 4-bit per pixel:
- **Portrait**: `0x4F, 0x68` (AM=1, scan Y direction)
- **Landscape**: `0x2F, 0x68` (AM=0, scan X direction)

### Reset Timing
Increased from 20ms to 50ms (both LOW and HIGH phases)
```c
gpio_set_level(PIN_RST, 0);
delay_ms(50);
gpio_set_level(PIN_RST, 1);
delay_ms(50);
```

### Post-Init Delay
Added 200ms delay after POWER ON command for panel stabilization

### Data Format
- 4-bit per pixel (2 pixels per byte, packed)
- Total bytes: 192000 (800 × 480 / 2)

## Watchdog Timeout Fix

### Root Cause
EPD clean cycle (white→black→white) was blocking too long, triggering task watchdog

### Solution
- Simplified to single black flash only
- Reduced yield frequency: 1000 → 500 pixels
- Setup mode now uses direct write instead of full refresh cycle

## BLE Configuration Transfer Fixes

### Chunk Size
Reduced from 180 bytes to 100 bytes per chunk

### Write Mode
Changed `withoutResponse: false` → `true` for faster transfer

### Timing
- Chunk delay: 250ms → 100ms
- Retry delay: 1000ms → 500ms
- Max retries: 3 → 5

### Timeout
Increased from 5s to 10s per chunk
